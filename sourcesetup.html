This page will guide you through setting up NadekoBot using the open source code.

When should I use this guide?

If you just want to use the bot without going through complicated things, use the normal Setup Guide.

Building from source is only handy when you want to modify your NadekoBot- as in adding commands, creating a new game, etc.

This guide is for using on Windows, and requires VisualStudio. While using other ways to build and compile the source may be possible, this guide does not go into that.
